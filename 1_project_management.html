Setting up a GitHub Account:



Step 1.
Go to www.github.com in your web browser.
Click the "Sign up" button in the top right corner.

Step 2.
Enter your email address.

Step 3.
Create a Strong Password.

Step 4.
Choose a Username.

Step 5.
You'll be prompted to verify your account, Verify it there.

Step 6.
Github will send a Launch Code to the Email accound you registered.

Step 7.
Go to your Email account and access the GitHub Launch code from there.

Step 8.
After this GitHub will ask you for some additional information, Answer accordingly.

Step 9.
Choose your GitHub Plan.

Step 10.
Github will Initialize and setup your Dashboard.

Step 11.
Click on "+" sign in the top right corner. in the dropdown choose Import Repository.

Step 12.
Go to this link to access Fabacademy Design Template or you can use any other template of your choice.

Step 13.
Copy Paste template's URL into your import repository page, You can choose whether you want to keep it public or private.

Step 14.
Click on "Begin imort" to start importing.

Step 15.
While repo is importing download and install GitHub Desktop for system which you are using

Step 16.
Also download any Code Editor of your choice, I installed Visual Studio Code because of its integration with GitHub.

Step 17.
After setting-up GitHub Desktop, On GitHub website navigate to repository you created, click on green code button, in the drop down select 'Https' and copy that link.

Step 18.
On GitHub Desktop click on 'Add'in the top left corner, choose "clone a repository", Paste URL link in the URL section and hit 'Clone'. Your repo will get cloned locally and you'll be prompted to a screen like this, choose open in Visual Studio Code here.

Step 19.
Now you can start customizing and editing your site in VS code.

Step 20.
You can add a readme file and copy paste the student agreement you received.
